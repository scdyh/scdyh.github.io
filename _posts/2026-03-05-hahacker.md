### 部署方式

我是从新建一个虚拟机并导入debian安装镜像开始部署的，这里不再赘述，注意只安装ssh不安装图形化即可

这里提供一些命令，方便搭建基础的环境



#### 1. 更新apt

```
apt update && apt upgrade -y
apt dist-upgrade -y
```

#### 2. 设置主机名（根据靶机名称）

```
echo "hAhacker" > /etc/hostname
hostnamectl set-hostname hAhacker
```

修改 `/etc/hosts`：

```
nano /etc/hosts
```

修改为：

```
127.0.0.1    localhost
127.0.1.1    hAhacker
```

#### 3. 配置时区和 NTP

```
timedatectl set-timezone Asia/Shanghai  # 根据你的时区修改
apt install ntp -y
systemctl enable ntp
systemctl start ntp
```

#### 4. 创建用户（hacker 用户）

```
adduser hacker
usermod -aG sudo hacker
```

为 `hacker` 用户设置密码：

```
passwd hacker
```

#### 5. 安装基础服务和工具

```
apt install -y sudo curl wget git vim net-tools lsof iputils-ping unzip iproute2 \
    ssh openssh-server sudo systemd
```

启用 SSH 服务：

```
systemctl enable ssh
systemctl start ssh
```



### 清理环境，确认权限

#### 1. 禁止玩家查看历史

```
echo 'export HISTFILE=/dev/null' >> /home/hacker/.bashrc
echo 'export HISTFILE=/dev/null' >> /root/.bashrc
```

#### 2. 清理 bash 历史记录

```
history -c
rm -f /root/.bash_history
rm -f /home/hacker/.bash_history
ln -sf /dev/null /root/.bash_history
ln -sf /dev/null /home/hacker/.bash_history
```

#### 3. 清理日志文件

```
rm -f /var/log/*.gz
rm -f /var/log/*.[0-9]
truncate -s 0 /var/log/syslog
truncate -s 0 /var/log/auth.log
```

#### 4. 设置用户的 shell 权限（避免脚本被修改）

```
chmod 700 /home/hacker
chmod 700 /home/hacker/.bashrc
```

#### 5. 配置 sudo 权限

```
echo "hacker ALL=(root) NOPASSWD: /usr/local/sbin/adbridge-maint" > /etc/sudoers.d/hacker
chmod 440 /etc/sudoers.d/hacker
```
