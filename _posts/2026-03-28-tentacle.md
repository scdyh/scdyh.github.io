这是 **Apache 的用户主页映射功能**。

访问的是：

```
http://$IP/~operator/Tentacle/app.py
```

这个 `~operator` 在 Apache 配置里被解释成：

```
/home/operator/public_html/
```

也就是用户 `operator` 的个人网站目录。

而前面本地已经看到过：

```
/home/operator/public_html/Tentacle/app.py
```

所以这个 URL 会自然映射到：

```
/home/operator/public_html/Tentacle/app.py
```

然后 Apache 直接把这个文件当静态文件返回给你了。

所以为什么会有 /home/licksore/public_html，就是 licksore 有一个类似的用户主页，所以也不应该是猜的
