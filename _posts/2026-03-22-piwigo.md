提示要爆破，那就尝试爆一下密码，字典用的是 `SecLists\Passwords\Common-Credentials\10-million-password-list-top-1000.txt`

![image-20260322215435056](/assets/img/posts/piwigo/image-20260322215435056.png)

弹个shell，在 /opt 下发现了一个 keepass 的数据库，爆破一下密码

```bash
while IFS= read -r p; do
  printf '%s\n' "$p" | keepassxc-cli ls my_database.kdbx >/dev/null 2>&1 && {
    echo "[+] FOUND: $p"
    break
  }
done < /usr/share/wordlists/rockyou.txt
```
